# Ucal - console unsure calculator

Inspired by [an awesome little article](https://filiph.github.io/unsure/) from Filip Hracek. You can get the original version from [his repo](https://github.com/filiph/unsure).

This is a console notebook-like calculator app for operations over a range of numbers. Read the article above for clarification of the original idea and examples of practical usage.

> [!NOTE]
> The project is at an early stage of development.
